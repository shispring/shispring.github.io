Kubernetes Scheduler Framework 一览表

阶段 作用 典型插件 是否可失败 失败后处理
PreEnqueue 是否允许进入调度队列 PodTopologySpread 是 Pod进入unschedulable list
QueueSort 队列排序 PrioritySort 否 仅排序
PreFilter 预检查/预计算 NodeResourcesFit 是 直接终止调度
Filter 过滤不可调度节点 TaintToleration 是 Node被淘汰
PostFilter 无节点时补救 DefaultPreemption 是 抢占失败则回队列
PreScore 打分前准备共享状态 InterPodAffinity 是 调度失败
Score 节点打分 NodeResourcesBalancedAllocation 是 Node分数无效
NormalizeScore 分数归一化 自定义插件 是 分数修正失败
Reserve 资源预占位 VolumeBinding 是 调用Unreserve
Permit 等待/批准/拒绝 GangScheduling 是 回滚+重新调度
PreBind 绑定前操作 VolumeBinding 是 回滚资源
Bind 真正绑定 DefaultBinder 是 回滚资源
PostBind 收尾清理 自定义插件 否 仅通知
Unreserve 回滚Reserve状态 VolumeBinding 否（必须成功） 清理状态
