3 openstack security group show 检查安全组规则
4 ping / telnet 验证网络连通性
5 traceroute 分析网络路径

4.4 Port 绑定异常

常见问题

问题 原因 解决方案
Port 状态为 DOWN 未绑定实例 绑定到 Server
无 Fixed IP 未分配 IP 重新创建或分配
安全组缺失 未正确绑定 重新设置安全组

4.5 跨 Region 资源问题

排查要点

问题 原因
找不到资源 查询的 Region 不正确
UUID 不一致 不同 Region 资源独立

5. ✅ 最佳实践总结

5.1 命令使用最佳实践

实践 说明
使用 -f yaml/json 便于自动化与脚本解析
使用 -c 指定字段 提高命令输出可读性
使用 --os-region-name 避免 Region 混淆
通过 ID 操作资源 避免名称重复问题

6. 📌 常用排查命令速查表

场景 命令
查看当前 Region echo $OS_REGION_NAME
查询 IP 归属 openstack port list --fixed-ip ip-address=<IP>
查看 FIP 绑定 openstack floating ip show <FIP>
查看端口状态 openstack port show <PORT_ID>
查看安全组规则 openstack security group rule list <SG_ID>
查看实例日志 openstack console log show <SERVER_ID>
指定 Region 查询 openstack --os-region-name <Region> server list
